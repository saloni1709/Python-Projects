# 🐍 Python Mini Projects

This repository contains two Python command-line projects:

- 🧮 Calculator with History  
- 🎮 Password Guessing Game  

---

## 🧮 Calculator with History

### 📌 Overview  
A simple calculator that performs basic arithmetic operations and stores history in a text file.

### ▶️ Run
```
python calculator_with_history.py
```

### 💻 Usage
```
2 + 2
10 - 5
6 * 3
8 / 2
```

### 📂 Commands
```
history  - show history
clear    - clear history
exit     - exit program
```

---

## 🎮 Password Guessing Game

### 📌 Overview  
A game where you guess a randomly selected word using hints.

### ▶️ Run
```
python guessing_game.py
```

### 🎯 How to Play
- Choose difficulty: easy / medium / hard  
- Guess the word  
- Use hints (_) to find correct letters  

### 📊 Example
```
Secret word: apple

Guess: appls
Hint: appl_
```
